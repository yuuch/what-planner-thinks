## 本地预览
需要安装 [mdbook](https://github.com/rust-lang/mdBook) 及其 [mermaid 插件](https://github.com/badboy/mdbook-mermaid)：

```bash
# 安装工具
cargo install mdbook
cargo install mdbook-mermaid

# 初始化 mermaid 支持 (仅需一次)
mdbook-mermaid install .

# 启动预览
mdbook serve --open
```
